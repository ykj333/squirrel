squirrel_finder.htm

주의 사항:

이미지 파일: koala.png, strawberry.png, squirrel.png 이미지 파일(각각 40x40 픽셀)을 이 HTML 파일과 동일한 디렉터리에 저장하세요.
레트로 색상 및 어두운 배경: 게임은 어두운 배경 색상 (#0A0A0A)과 레트로 스타일의 노란색 (#FFFF00)을 사용합니다.
조작법: 화살표 키를 사용하여 코알라를 빠르게 이동할 수 있습니다.
딸기: 매 초마다 생성되며 화면에서 돌아다닙니다. 부딪히면 게임이 종료됩니다.
다람쥐: 3초 후에 나타나며 부딪히면 게임에서 승리합니다.
"openai"와 타이머: 화면 상단 중앙에 "openai" 텍스트가 표시되고, 좌측 상단에 타이머가 표시됩니다.
게임 시작 전 설명: 게임 시작 전에 설명이 표시되며, 아무 키나 누르면 게임이 시작됩니다.
자동 재시작: 승리하거나 패배한 후 자동으로 게임이 재시작됩니다.
실행 방법:

위의 코드를 예를 들어 squirrel_finder.html 파일로 저장하세요.
필요한 이미지를 동일한 디렉터리에 저장하세요.
웹 브라우저로 squirrel_finder.html 파일을 열면 게임을 플레이할 수 있습니다.
즐겁게 "Squirrel Finder"를 플레이하세요!
